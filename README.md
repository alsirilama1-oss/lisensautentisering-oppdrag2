# lisensautentisering-oppdrag2# 

Dette prosjektet er en del av faget Datasikkerhet.

## Hva gjør programmet?
Programmet ber brukeren om en lisenskode.
Hvis koden er riktig, får brukeren tilgang.

## Teknologi
- Python
- VS Code

## Hvordan kjøre programmet?
python program.py


##Flask og lisenssjekk
Programmet sender lisenskoden til en lokal Flask-server som sjekker om lisensen er gyldig før programmet får starte.
